GeoServer Connection Notes

The placeholder blocks already include empty attributes for future GeoServer integration. When the final WMS or image URLs are available, fill in data-geoserver-url, data-layer, data-style, and image source values.

Note: This project should not use local PNG files or local file paths such as file:// or C:\.... Final map images should be loaded from GeoServer WMS or another public web URL so the page works after it is published on GitHub Pages.
