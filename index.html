Three Independent Pollutant Maps

The largest categories are NO2, PM2.5, and PM10. Each category has one full-width map: choose the basemap first, then choose which thematic layer appears above it.

NO2 Nitrogen dioxide One basemap, selectable overlay layer. PM2.5 Fine particulate matter One basemap, selectable overlay layer. PM10 Coarse particulate matter One basemap, selectable overlay layer.

GeoServer Connection Notes

Each pollutant map has one basemap and one selectable overlay layer. To connect a layer, put the WMS endpoint in data-geoserver-url and the full workspace-qualified layer name in data-layer on the corresponding dropdown option.

Pollutant Map behavior Layer choices
NO2 One full-width basemap Pollution / Land Cover / Population placeholders
PM2.5 One full-width basemap Pollution / Land Cover / Population placeholders
PM10 One full-width basemap Four GeoServer pollution layers plus land cover and population placeholders
